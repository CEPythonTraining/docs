Embedded System#

In weeks 2 and 3, a Raspberry Pi 4 and a Raspberry Pi Camera Board V2 - 8MP is used.

To operate the Raspberry Pi, it is needed to arrange for a Raspberry Pi power supply, micro-SD card and micro-SD card USB-adapter and cooling. Use a fast micro-SD card for optimal performance. The cooling can be either passive or active. For initial usage, you‚Äôll need a keyboard, a mouse, and a display. It is assumed that you have such a set available.

Infrastructure#

A Git-repository is required for some Git exercises and to store all created code. The Git-repository is located on a Sogeti hosted server and accessible through VPN or via the office network.
